## `ls`
Lists information about files

* `-a`: do not ignore entries starting with `.`
* `-A`: do not list implied `.` and `..`
* `-h`: human-readable file sizes
* `-l`: long listing format
* `-r`: reverse order
* `-t`: sort by time, newest first
* `-1`: list one file per line
